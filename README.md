## Usage

1. Clone the repository:
  ```
  git clone git@github.com:Nacho-Wifi/pearl.git
  cd pearl
  ```

2. Create a `.env` file in the root to store your API keys:

3. Install dependencies:
    ```
    npm install
    ```
4. Start expo to run simulator:
    ```
    expo start
    ```
